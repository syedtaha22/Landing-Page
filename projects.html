Chess Engine

This chess engine, built in C++, utilizes sophisticated algorithms to play chess effectively. It incorporates a minimax algorithm with alpha-beta pruning for optimal move decisions. The engine evaluates positions using heuristic functions and supports various difficulty levels and custom settings. This project highlights advanced skills in data structures, algorithms, and performance optimization. Through iterative testing and refinement, this engine aims to provide a challenging and engaging experience for chess enthusiasts and players of all skill levels.

GAN for Nebula Images

This Generative Adversarial Network (GAN) generates high-quality nebula images. Leveraging TensorFlow and Keras, the GAN consists of a generator for image creation and a discriminator for evaluating authenticity. Through iterative training, the GAN produces stunning nebula visuals, showcasing AI's potential in creative domains and deep learning expertise. This project involves advanced techniques in neural network training and image generation, offering insights into the power of GANs in producing realistic and artistic representations.

FlashAI

FlashAI is a comprehensive website for generating flashcards, featuring AI-driven suggestions. Developed with Next.js, React, Material UI, and Firebase, it uses the OpenAI API for smart content generation. FlashAI simplifies study management and enhances learning efficiency. The application offers a seamless user experience, integrating modern web technologies to streamline flashcard generation and optimize study sessions. Through intuitive design and powerful backend functionality, FlashAI is a practical tool for learners.

Key Achievements:

  • Created an intuitive UI using Next.js, React, and Material UI for a user-friendly experience.
  • Integrated Firebase for real-time data management and secure user authentication.
  • Ensured mobile and desktop responsiveness to provide a consistent user experience across devices.
  • Implemented AI-based flashcard suggestions via the OpenAI API, enhancing study session efficiency.
  • Developed features for effective study management, aiming to optimize learning and retention.
